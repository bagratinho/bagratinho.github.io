Mission

Create the world's largest repository of public and private data. Information Garden provisions structured data for use and sharing with the main focus to facilitate collaboration for research, business, scientific pursuits and academic efforts. (Cognate: GitHub for data/StackOverflow for analysis)

How

This is accomplished by providing users with the ability to access and create repositories of data including; public data sets which have been vetted and formatted as well as user generated structured information (tables, JSON and CSV). Repositories can be shared with others and a robust comment/analysis section in each repository enables users to share analysis.

Model

Freely available public data is combined with technology for upload/download/API providing the ability to share or store data publicly or privately. Open data repositories are free, private repositories are paid.
